# Notes on Bit Picking


> **_IMPORTANT_**

* Shifting by a negative number or a number greater than bit width is undefined!
* All vacant bits are filled in by 0.
* Shifting by 0 always results in 1:

<p style="margin-left:80px">
```c
1 << 0 = 1 because x^1 == 1
```
